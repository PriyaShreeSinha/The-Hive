THE HIVE 🐝
===========
A study spot reservation app built for UMass Boston students.

-----------
DESCRIPTION
-----------
The Hive helps UMass Boston students find and reserve quality study spots
on campus. Users can filter by noise level, seat availability, and outlet
access, then reserve a spot, get directions, and check in/out via QR code.

--------
FEATURES
--------
- Splash / Welcome screen
- UMB login (email + password or UMB credentials)
- Feature filters: Noise Level, Seat Availability, Outlet Availability, Snacks Near Me
- Honey Spot map powered by Google Maps Embed API
- Reserve a Spot with date/time picker and session goal
- Reservation confirmation screen
- Reminder notification screen
- Directions map to University Hall, UMass Boston
- QR code check-in and check-out screens
- Notifications screen
- End of session screen
- Post-session survey with star ratings
- Animated slide-out hamburger menu with dropdowns

------
SETUP
------
1. Download or clone the repository.
2. Open index.html in any modern web browser — no server required.
3. The Google Maps Embed API key is already configured.
   If the map stops working, replace the key in index.html by searching for
   "AIzaSyC0Q6uN7-3GTZfl8bIhe5n37xF4fo-jGig" and substituting your new key.
